# 🗺️ [Master Plan] 파이썬 알고리즘 정복 로드맵

*(총 5단계 / 약 3~4개월 코스)*

### **STEP 0. Pythonic_Basic**

### **STEP 1. 워밍업: "R 물 빼기 & 파이썬 뇌 장착"**

- **목표:** R의 `vector` 사고방식에서 벗어나, Python의 `List`, `Slicing` 등 기본 문법의 **'숏코딩(Short-coding)'** 맛을 본다.
- **교재 챕터:**
    - **6장: 문자열 조작** (팰린드롬, 로그 파일 재정렬)
    - **7장: 배열** (두 수의 합 - *이건 무조건 풀어야 함*)
- **통계적 의미:** 데이터 전처리(Preprocessing)와 동일. 텍스트 데이터 클렌징하는 로직.

### **STEP 2. 자료구조 1: "속도의 핵심, $O(1)$을 찾아라"**

- **목표:** **해시(Dictionary)**를 자유자재로 다루기. 코테 효율성의 80%는 여기서 판가름 남.
- **교재 챕터:**
    - **11장: 해시 테이블** (생일 문제, 보석과 돌)
    - **9장: 스택, 큐** (괄호 유효성 검사 - *스택의 기본*)
    - **(8장 연결 리스트는 건너뜀 - 파이썬에선 효율성 떨어짐. 나중에 시간 남으면 봐)**
- **통계적 의미:** Key-Value 매핑. 함수 $y=f(x)$ 구현. 검색 속도를 $O(N)$에서 $O(1)$로 줄이는 마법.

### **STEP 3. 자료구조 2: "공간과 계층을 다루는 힘" (★GPI 필수)**

- **목표:** **트리(Tree)**와 **그래프(Graph)** 정복. 네가 만들 **GPI(공간통계)** 라이브러리의 핵심 로직(KD-Tree 등)을 이해하기 위한 필수 구간. **여기가 제일 중요함.**
- **교재 챕터:**
    - **12장: 그래프** (DFS/BFS - *섬의 개수 구하기*)
    - **14장: 트리** (이진 트리 반전, 최대 깊이)
    - **15장: 힙(Heap)** (배열의 K번째 큰 요소 - *우선순위 큐*)
- **통계적 의미:** 군집화(Clustering), 의사결정나무(Decision Tree), 최단 경로 탐색.

### **STEP 4. 알고리즘: "필터링과 탐색의 미학"**

- **목표:** 데이터를 정렬하고 빠르게 찾는 법. **'이진 탐색'**은 대용량 데이터 처리의 기본 소양.
- **교재 챕터:**
    - **17장: 정렬** (퀵 정렬, 병합 정렬 개념만 잡기)
    - **18장: 이진 검색** (입국 심사 - *이거 진짜 좋은 문제*)
    - **20장: 슬라이딩 윈도우** (투 포인터 개념)
- **통계적 의미:** 분위수(Quantile) 찾기, 구간 추정. Time Complexity($O(N \log N)$)에 대한 감각 익히기.

### **STEP 5. 보스전: "최적화의 끝판왕, DP" (몰로코 합격선)**

- **목표:** 점화식을 세워서 최적해를 찾는 연습. 어렵지만 여기를 넘어야 Tech 기업 문이 열림.
- **교재 챕터:**
    - **23장: 다이나믹 프로그래밍(DP)** (계단 오르기, 0/1 배낭 문제)
    - **22장: 분할 정복**
- **통계적 의미:** 베이지안 갱신(Update), 마르코프 체인, 최적화 문제(Optimization).
